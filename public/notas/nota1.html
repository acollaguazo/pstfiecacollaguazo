8 192.168.2.36/30 Aquí existe un espacio libre entre las subredes

         11000000.10101000.00000010.00100100 <- IP de Subred 8
AND 11111111.11111111.11111111.11110000 <- Máscara en binario
         11000000.10101000.00000010.00100000 <- IP de Subred 8 corregida
Subred 8 corregida -> 192.168.2.32/28
Saltos de 24=16 en el cuarto octeto la siguiente subred sería 192.168.2.48

Se realiza la operación AND para obtener la Subred correcta
7 192.168.2.100/30          11000000.10101000.00000010.01100100 <- IP de Subred 9
AND 11111111.11111111.11111111.11000000 <- Máscara en binario
         11000000.10101000.00000010.01000000 <- IP de Subred 9 corregida
Subred 9 corregida -> 192.168.2.64/26
